pour déclarer un array on fait 
things = ['a', 'b', 'c', 'd']

things[0] = a
things[2] = c

un 'Hash' est un array ou on peut associer n'importe quelle pair d'objets ensemble

pour déclarer un Hash
stuff {"name"=>"Alexandre", "age"=>25, "height"=>193}
//simple cote ou double cote au choix

pour acceder à la taille on fait stuff['height']

Si 'stuff' est un 'Hash' contenant une clef 'pouet' -> stuff.delete('pouet') pour delete
et on peut facilement rajouter des "clefs" genre stuff[1] = "Wow" se rajoutera en bout de Hash

EXEMPLE :

# Create a mapping of state to abbreviation
states = 
{
'Oregon' => 'OR',
'Florida' => 'FL',
'California' => 'CA',
'New York' => 'NY',
'Michigan' => 'MI',
}

# create a basic set of states and some cities in them
cities =
{
	'CA' => 'San Francisco',
	'MI' => 'Detroit',
	'FL' => 'Jacksonville'
}

# add some more cities
cities['NY'] = 'New York'
cities['OR'] = 'Portland'

#puts out some cities
puts '-' * 10
puts "NY State has: #{cities['NY']}"
puts "OR State has: #{cities['OR']}"

#puts some state
puts '-' * 10
puts "Michigan's abbreviation is : #{states[´Michigan´]}"
puts "Florida's abbreviation is : #{states['Florida']}"

#do it by using the state then cities dict
puts '-' * 10
puts "Michigan has : #{cities[states['Michigan']]}"
puts "Florida has : #cities[states['Florida']]}"

#puts every state abbreviation
